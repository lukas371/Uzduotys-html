1. Svetaines URL. Kas sudaro svetaines URL-a. (kaip pvz. https://www.w3schools.com/php/showphp.asp?filename=demo_datatypes_null, t.y. kas yra https:// ; www ir t.t.)
URL yra universalus informacijos šaltinio adresas HTTPS(TLS apsaugos standartas su idiegtu SSL sertifikatu) yra saugi HTTP versija (pagrindinis metodas informacijai pasauliniame tinkle (WWW) pasiekti.
Kai lankytojas ateina i svetaine HTTPS adresu, serveris jam atsiuncia vieša SSL sertifikato dali. 
Pagal ši sertifikata naršykle patikrina, ar SSL sertifikata tikrai išdave patikima sertifikavimo institucija, kuri nurodyta prie rakto.
2. PHP programavimo kalbos paskirtis. (kam ji sukurta, kur naudojama, papildomai: kokios perspektyvos?)
PHP programavimo kalba yra specialiai pritaikyta interneto svetainiu kurimui .Sukurti galima daugybe dalyku, nuo statines kontaktines svetaines su kontaktu forma, iki sandelio/buhalterijos valdymo sistemu. 
Nuo svetaines lankomumo skaitliuko iki web browsiniu žaidimu. 
Kadangi kodas talpinamas serveryje, galima sukurti sistemas valdomas mobiliaisiais irenginiais per naršykles, išvengiant papildomo programavimo papildomiems irenginiams.
80% visu internetiniu puslapiu back-end yra suprogramuota PHP programavimo kalbos, todel laukia laisvai samdomo specialisto ar tinklalapiu kurimo agenturos karjera.
3. Išvardinkite pagrindinius duomenu tipus ir pateikite ju pavyzdžius (be object ir resource, t.y. 6. Duomtipas - apibudinimas - pavyzdys).
Boolean (logine reikšme)	true	Viena iš specialiuju reikšmiu true arba false
Integer (sveikasis skaicius)	10	Sveikasis skaicius
Float arba Double	4,6842	slankiojo kablelio skaicius
String (eilute)	"sveiki, maze lankytojai"	Simboliu rinkinys
Array (masyvas) tipas yra vienas iš daugiausiai naudojamu tipu PHP kalboje. Su masyvu, galima i viena masyva ideti daugiau negu viena kintamaji grupavimo ir panašiam tikslui.
NULL tipas reiskia nieko, arba nuli. null
4.Turime du kintamuosius int tipo: $a ir $b.
 Jiems yra priskirtos reikšmes.

kintamasis a yra didesnis už kintamaji b ir analogiškai del kitu situaciju. Pateikite savo sprendima. (salygos sakinys IF, teksto išvedimas ECHO, papildomai - jei a===b)
